## ✅ VERSÃO 3: POST LONGO (Storytelling + Técnico)

```
🧠 Lançamento: CLAW v1.1.4 — A Extensão VS Code que se adapta a SUA IA

**TL;DR:**
Criei uma extensão VS Code que detecta automaticamente qual IA você tem e oferece sugestões inteligentes. 5-10x mais rápido que o padrão. 99% mais barato.

Baixe aqui: https://marketplace.visualstudio.com/items?itemName=RafaelBatista.clawrafaelia-suggestions

---

**A HISTÓRIA:**

Há alguns meses eu estava frustrado com as limitações das extensões de IA para VS Code. Todas achavam que você estava preso a uma IA específica.

Mudou de Gemini para OpenAI? Reconfiguração manual.
Quer tentar Claude? Mais reconfiguração.
Tem LocalAI rodando localmente? Esqueça.

**"Por que não existia uma extensão que simplesmente reconhecesse o que você tem e funcionasse?"**

Decidi criar uma.

---

**COMO FUNCIONA:**

Quando você abre VS Code com CLAW:

1. **AIProbe** — Testa em paralelo: Gemini? OpenAI? Claude? LocalAI? Ollama?
2. **AISelector** — Escolhe a melhor disponível
3. **SmartFallback** — Se uma falha, tenta a próxima
4. **Cache Semântico** — Reusar sugestões similares (ZERO API calls)
5. **Fallback Local** — Mesmo sem IA, funciona com patterns + templates

Resultado? Você digita → sugestões aparecem → pressiona Tab → done ✅
```
